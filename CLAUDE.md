# tamudashe.github.io

Photography portfolio site at tamudashe.com, hosted on GitHub Pages.

## Structure

- `photos/{category}/` — full-res originals
- `photos/thumbs/{category}/` — thumbnails generated by `generate.sh` via `sips -Z 1200`
- `photos.json` — manifest with `_all` (global ranking) and per-category arrays
- `generate.sh` — generates missing thumbnails and updates `photos.json`

## photos.json format

```json
{
  "_all": ["category/filename.jpg", ...],
  "category": ["filename.jpg", ...]
}
```

`_all` controls order in the All view (best photos first). Per-category arrays control order in filtered views. `generate.sh` preserves existing order and appends new photos.

## Workflow

1. Drop photos into `photos/{category}/`
2. Run `./generate.sh`
3. Edit `photos.json` to reorder as needed (best first)
4. `git add -A && git commit && git push`

## Design

- Apple design language: SF Pro font, `#fff` / `#1d1d1f`, `border-radius: 980px` pill shapes
- CSS masonry grid: `columns: 3`, 8px gaps
- Black lightbox: `rgba(0,0,0,0.95)` with prev/next navigation
- Photo fade-in via `.loaded` class on `img` load event
- URL hash routing: `#category` syncs with active filter
- Frosted glass scroll-to-top button: `backdrop-filter: blur(20px) saturate(180%)`
- Cache bust: `script.js?v=N` — increment N when deploying JS changes
